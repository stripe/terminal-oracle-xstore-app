# Stripe Connector for Oracle Xstore

This repository hosts release binaries for the [Stripe Connector for Oracle Xstore](https://docs.corp.stripe.com/use-stripe-apps/oracle/xstore/overview).

> **Important:** This connector is not a self-service solution. Installation and setup require direct engagement with a Stripe account team representative (e.g., Solutions Architect, Account Executive Partner, or Professional Services).
>
> If you are interested in using this connector, please contact your Stripe account team for guidance and support.
